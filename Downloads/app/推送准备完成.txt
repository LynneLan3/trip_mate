🔐 身份验证方式

【使用 GitHub Token (HTTPS)】

  1. 生成 token:
     https://github.com/settings/tokens

  2. 创建 Personal Access Token
     - Scope: repo (全部)
     - 复制生成的 token

  3. 推送时使用 token:
     - 当提示输入密码时, 粘贴你的 token

【使用 SSH 密钥】

  1. 检查 SSH 密钥:
     ls -la ~/.ssh/

  2. 如果没有，生成新密钥:
     ssh-keygen -t ed25519 -C "你的邮箱"

  3. 添加到 GitHub:
     https://github.com/settings/ssh

  4. 测试连接:
     ssh -T git@github.com
