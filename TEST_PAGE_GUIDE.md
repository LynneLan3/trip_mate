## ⚠️ 可能遇到的问题

### 问题1: 显示"❌ 连接失败"
**原因**: 
- 网络连接问题
- Supabase服务不可用
- CORS策略限制

**解决方案**:
- 检查网络连接
- 刷新页面重试
- 查看浏览器控制台（F12）的错误信息

### 问题2: 加载时间很长（>30秒）
**原因**: 
- 网络速度慢
- Supabase服务响应慢

**解决方案**:
- 等待加载完成
- 刷新页面重试

### 问题3: 某些问卷显示"0 道题目"
**原因**: 
- 该问卷的题目数据加载失败
- 数据库中该问卷确实没有题目

**解决方案**:
- 检查浏览器控制台的错误信息
- 在Supabase Dashboard中检查数据

## 🔍 调试方法

### 查看浏览器控制台
1. 按 **F12** 打开开发者工具
2. 切换到 **Console** 标签页
3. 查看是否有红色错误信息

### 查看网络请求
1. 按 **F12** 打开开发者工具
2. 切换到 **Network** 标签页
3. 刷新页面
4. 查看 API 请求的状态码和响应

### 常见错误信息
```javascript
// 网络错误
Failed to fetch

// CORS 错误
Access to fetch at '...' from origin 'null' has been blocked by CORS policy

// 认证错误
401 Unauthorized

// 服务器错误
500 Internal Server Error
```
