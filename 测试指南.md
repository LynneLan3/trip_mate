## 🛠️ 如果遇到问题

### 问题1: 看不到问卷数据
**解决方案**: 执行数据库SQL脚本
1. 打开 Supabase Dashboard
2. 进入 SQL Editor
3. 执行 `添加问卷隐私字段.sql` 文件内容

### 问题2: 页面显示空白
**解决方案**:
- 检查浏览器控制台是否有错误
- 尝试硬刷新 (Cmd+Shift+R)
- 确认已登录账号

### 问题3: 编辑权限错误
**解决方案**:
- 确认是问卷创建者
- 检查数据库中 creator_id 字段是否正确设置

### 问题4: 公开问卷不显示在首页
**解决方案**:
- 检查问卷的 is_public 字段是否为 true
- 确认数据库SQL脚本已执行
