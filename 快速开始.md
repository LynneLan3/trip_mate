# ⚡ 快速开始指南

## 3步完成问卷导入

### 第1步：打开 Supabase SQL Editor

1. 访问 [Supabase Dashboard](https://app.supabase.com)
2. 选择你的项目
3. 点击左侧 **SQL Editor** 📝

### 第2步：执行 SQL 脚本

1. 点击 **New query** 按钮
2. 打开 `新增5个问卷类型.sql` 文件
3. 复制全部内容（Ctrl+A, Ctrl+C）
4. 粘贴到 SQL 编辑器（Ctrl+V）
5. 点击 **Run** 按钮 ▶️
6. 等待提示：`5个问卷类型及所有题目已成功创建！`

### 第3步：刷新应用

1. 打开你的应用首页
2. 强制刷新页面（Ctrl+F5 或 Cmd+Shift+R）
3. 看到5个新问卷卡片 🎉
